§7.3 A Minimal Counterexample to Distributivity in the Subspace Lattice

The concrete algebraic expression: within the subspace lattice of a complex Hilbert space, non-distributivity can be exhibited by a minimal example.

Consider the two-dimensional complex Hilbert space $\mathbb{C}^2$, with:

  • $A = \text{span}\{e_1\}$ (the one-dimensional subspace along $e_1$)
  • $B = \text{span}\{e_2\}$ (the one-dimensional subspace along $e_2$)
  • $C = \text{span}\{e_1 + e_2\}$ (the one-dimensional subspace along $e_1 + e_2$)

where $\{e_1, e_2\}$ is an orthonormal basis of $\mathbb{C}^2$.

In the subspace lattice, $\lor$ corresponds to span (linear combination) and $\land$ to intersection:

$$B \lor C = \text{span}\{e_2, e_1 + e_2\} = \text{span}\{e_1, e_2\} = \mathbb{C}^2$$

$$A \land (B \lor C) = A \land \mathbb{C}^2 = A = \text{span}\{e_1\}$$

On the other side:

$$A \land B = \text{span}\{e_1\} \cap \text{span}\{e_2\} = \{0\}$$

$$A \land C = \text{span}\{e_1\} \cap \text{span}\{e_1 + e_2\} = \{0\}$$

$$(A \land B) \lor (A \land C) = \{0\} \lor \{0\} = \{0\}$$

Hence:

$$A \land (B \lor C) = \text{span}\{e_1\} \neq \{0\} = (A \land B) \lor (A \land C)$$

Distributivity fails.

§3.5 C 类证伪条款的精确化

P1 §8.2 anchor 的 C 类（formalism failure）证伪条款：如果量子力学可被严格 reformulate 在纯实数本体地位上，SAE 复振幅本体身份需要 reconsider。

证伪条款的精确化辩护理据如下：实数 reformulation 之所以不可能完全替代，不是因为「复数更方便」或「数学习惯」，而是因为：

  • L₀→L₁ 单余项状态结构上无法承载 L₁→L₂ dualization 互锁关系（§3.1）
  • 任何成功的实数 reformulation 必然在某处引入额外结构来恢复 dualization 自由度（通常是引入两个实数自由度配对 $(x, y) \in \mathbb{R}^2$，且配对方式必须保持 $i$ 与 $\pi$ 互锁结构）
  • 这种引入额外结构的实数 reformulation 本质上仍然是复振幅本体的不同表述——它在数学形式上是实数 representation，在本体上仍然承载 L₁→L₂ dualization

但仅仅这样陈述还不够。任何能 reproduce 标准 QM 预言的 reformulation 都会在数学结构上等价于复振幅，SAE 总可以 argue "在某处仍 carry dualization"。为了让 C 类证伪条件具有实际可检验性，需要 articulate 更精确的判据：

SAE 主张 "carrying dualization" 的具体数学 indicator：reformulation 内部存在两个 algebraically independent 但 mutually dual 的 elements，使得它们的某个组合产生 $e^{i\pi}+1=0$ 类闭合方程结构（或与之等价的代数闭包关系）。

具体到已知 reformulation：

  • Stueckelberg real-Hilbert-space reformulation（Stueckelberg 1960）：在实希尔伯特空间上引入 $SO(2)$ rotation 作为额外结构，模拟复结构。$SO(2)$ 的生成元 $J$ 满足 $J^2 = -1$——这本身就是 $i$ 的代数 indicator（"algebraic root of $-1$"）。Stueckelberg formulation 因此在 algebraically independent dual element 这一指标上仍然 carry dualization。
  • Wootters real-vector-space program（Wootters 1990s）：在实向量空间上引入 complex structure $J$，同样满足 $J^2 = -1$——同样 carry dualization 的代数 indicator。

按上述判据，已知 reformulation 都没有证伪本文 articulation。

C 类证伪条件因此精确化为：如果存在某种 reformulation 完全基于无 dual algebraic structure 的代数对象（例如仅 abelian 加法群 over real numbers，无任何 multiplicative interaction 产生 $J^2 = -1$ 类元素），且能 reproduce 标准 QM 所有可观测预言（干涉、纠缠、Bell 不等式违反等），则 SAE 复振幅本体身份被证伪。此条件下 reformulation 必须显式不携带任何 $i$ 类 algebraic element 或与之等价的对偶结构，这是可代数检验的判据。

§4.2 离散 cell 与 compact internal fiber 二分

$\theta$ 的连续取值（$\theta \in [0, 2\pi)$）与 SAE 框架的 cell-based 离散本体之间存在一个表面张力：SAE cell 基底是 Planck-scale 离散的（继承《相对论 P1》§2.5），但 $\theta$ 看起来在连续区间上取值。

这一张力的 SAE 框架内 resolution：离散 base + compact internal fiber 二分。

  • Cell index discrete：cell 在 Planck 基底层 + 因果槽层上是离散的，cell 之间不连续地相邻（继承《相对论 P1》cell 双层结构）。
  • Cell internal fiber compact：每个 cell 上的 dualization coordinate $\theta$ 在 compact 内部 fiber 上取值。连续 $[0, 2\pi)$ 是该 fiber 的数学 completion，不是物理空间连续性。
  • 拓扑结构：fiber 的代数 closure 形式是 $U(1)$（单位圆周）；$\theta$ 沿 fiber 的位置由 dualization 关系的当前 instantiation 决定。

这一二分在数学结构上对应离散底空间上的 fiber bundle 模式：底空间（cell 集合）是离散的，但每个底点上的 fiber 是 compact 连续流形（$U(1)$ circle）。这一对象层结构在物理学中有多个 well-known 类比——lattice gauge theory 把 $U(1)$ gauge variable 放在 lattice links 上（site 之间的 link），与本文 fiber 放在 cell（site）本身上略有不同，但 base discrete + fiber compact 的总体对象层模式一致。SAE 不等同 lattice gauge theory，类比仅作对象层 illustration。

本文工作于 cell-level 离散本体；cell-aggregate 上的 emergent continuity（多 cell 聚合后呈现连续 $\psi$ 描述）由 P9 处理。

T3 candidate 标注：是否在更深 Planck / DD 层级该 fiber 退化为有限 root-of-unity 离散谱（例如 $\theta \in \{2\pi k/N : k = 0, 1, \ldots, N-1\}$ 某个有限 $N$），是一个尚未确定的开放问题，留作量子化与谱结构问题的 T3 candidate。如果未来 quantum gravity 工作给出某种 finite phase spectrum，与本文框架兼容；如果给出严格 continuous phase spectrum，也与本文框架兼容。本文在此问题上保持 ontological neutrality。

§4.4 θ = π 作为 closure-invocation locus

当 $\theta = \pi$ 时，复振幅 $z = A \cdot e^{i\theta}$ 取值 $z = -A$，闭合方程 $e^{i\pi}+1=0$ 在该 fiber 位置 algebraically manifest。

关于 $\theta = \pi$ 这一 special locus 的本体身份需要做几点澄清，以避免与后续论文议题混淆。

第一，$\theta = \pi$ 是 fiber 上闭合方程 $e^{i\pi}+1=0$ 在代数层级显式 manifest 的特定 locus，让 dualization 关系的代数互锁结构在该位置可见。Closure events 不限于在 $\theta = \pi$ 位置发生——闭合事件可以在 $\theta$ 的不同位置由不同 trigger 启动；$\theta = \pi$ 是闭合方程在 fiber 上的代数 anchor，不是闭合事件发生的唯一位置。

第二，$\theta = \pi$ 不应读作"测量塌缩"。测量作为 4DD ρ-AND 闭合事件的本体身份由 P7 处理。结构上预备进入闭合事件的位置（fiber 上的特定 locus），与闭合事件本身的发生（4DD ρ-AND swap-class event），是两件不同的事。

第三，$\theta = \pi$ 不直接触发概率读出。模平方读出的结构起源由 P6 处理。

第四，$\theta = \pi$ 不自动导致 closure。具体何时、如何由此进入 4DD ρ-AND 闭合事件需要外部触发条件；相关讨论由 P6 / P7 承担。

综合以上，$\theta = \pi$ 在本文中的身份是闭合方程 $e^{i\pi}+1=0$ 在 fiber 上的代数 anchor locus——一个结构层面的特定位置，与闭合事件本身的发生不等同。

§4.6 双尺度：cell-internal vs cell-aggregate relative phase

$\theta$ 在两个尺度上各自有本体身份：

Cell-internal $\theta_c$：每个 cell $c$ 携带自己的 $\theta_c$。这是 §4.1-§4.5 处理的对象——单 cell 在 compact $U(1)$ fiber 上的静态位置参数。Cell-internal $\theta_c$ 的物理内容是该 cell 内部 1DD 标识层的 dualization 关系位置。

Cell-aggregate relative phase：cell 聚合体上的物理内容主要在相对相位结构——

$$\Delta\theta_{c_1, c_2} = \theta_{c_1} - \theta_{c_2}$$

这是 cell aggregate 层级的 substantive 内容。绝对 $\theta_c$ 在 cell-internal level 有 anchor（$U(1)$ fiber），但 cell 之间的物理相干性主要由相对相位结构承载。

但这一相对相位的 well-definedness 本身需要 articulate。每个 cell 的 $U(1)$ dualization fiber 是 internal 的——若两个 cell 之间不存在共享的拓扑背景，"$\theta_{c_1}$ 与 $\theta_{c_2}$ 的差" 在数学上是不良定义的（两个独立 $U(1)$ fiber 之间的 phase 差需要 connection 才能给出客观意义）。

SAE 框架在此处的 anchor 来自《相对论 P1》（DOI: 10.5281/zenodo.19836185）§2.5 已立的 cell 双层结构：Planck 基底层（sub-causal 绝对层）与因果槽层。Planck 基底层作为绝对刚性背景，为所有 cell 提供共享的拓扑参照——cells 内部的 $U(1)$ fiber 通过 Planck 基底层的绝对平移联络（parallel transport over Planck base layer）实现跨 cell 对齐。$\Delta\theta_{c_1, c_2}$ 因此良定义为：cell $c_2$ 处的 fiber 位置 minus cell $c_1$ 处的 fiber 位置（经 Planck 基底层联络平移到同一参照下）。

这一 anchor 对 P5 纠缠 articulation 至关重要——纠缠 cells 之间的 relative phase 一致性正需要这个共享 Planck 基底层联络作为本体基础。

这一双尺度区分的重要性：

  • P5（纠缠）重度依赖 cell-aggregate relative phase 结构——本节为 P5 提供基础
  • 整体相位（所有 cell 同步乘以共同相位）是 dualization 关系下的 gauge-like redundancy（详 §8 SU(n) 同源关系）
  • 干涉、隧穿、纠缠等量子相干现象的本体载体是 relative phase pattern，不是 absolute $\theta_c$

双尺度区分至此立稳。纠缠的完整 articulation 见 P5，干涉的动力学见 P3 与 P4。

§6.2 cells 集合上多 dualization 状态的并存

ρ-OR 多容并存在 cell-indexed assignment 上的本体载体表达：

考虑一个 cell 集合 $C$，每个 cell $c$ 携带自己的复振幅 $z_c = A_c \cdot e^{i\theta_c}$。前闭合 ρ-OR 状态下，$z_c$ 不必处于「确定单一」状态——它可以作为多个 dualization 状态的并存。

具体地，若一个 cell $c$ 在前闭合 ρ-OR 状态下同时可能携带 $z_c^{(1)} = A^{(1)} \cdot e^{i\theta^{(1)}}$ 与 $z_c^{(2)} = A^{(2)} \cdot e^{i\theta^{(2)}}$ 这两个 dualization 状态，cell $c$ 的本体表达是这两个状态在前闭合层级的并存。

这一并存在 cell-indexed assignment 上的具体形式：

$$z_c = z_c^{(1)} + z_c^{(2)} = A^{(1)} \cdot e^{i\theta^{(1)}} + A^{(2)} \cdot e^{i\theta^{(2)}}$$

注意这里的 $+$ 不是「两个数相加得第三个数」，而是 dualization 状态在 ρ-OR realm 中的并存表达。两个 dualization 状态在 4DD ρ-AND 闭合之前真实并存，不互相排斥；它们在 cell 上以复振幅之和的形式呈现。

扩展到多 dualization 状态的并存：

$$z_c = \sum_k z_c^{(k)} = \sum_k A^{(k)} \cdot e^{i\theta^{(k)}}$$

这是单个 cell 上多容并存的载体形式。

扩展到 cell 集合上：每个 cell 各自携带多 dualization 状态的并存，整个 cell-aggregate 的 ψ 是这些 cell-level 并存的 cell-indexed assignment：

$$\Psi(c) = \sum_k z_c^{(k)} = \sum_k A^{(k)}_c \cdot e^{i\theta^{(k)}_c}$$

或者把多 dualization 状态分解为多个独立的 cell-indexed assignment $\Psi^{(k)}: C \to \mathbb{C}$：

$$\Psi = \sum_k \Psi^{(k)}, \quad \Psi^{(k)}(c) = z_c^{(k)} = A^{(k)}_c \cdot e^{i\theta^{(k)}_c}$$

这就是 ψ 作为多 dualization 状态的并存的载体表达。

§6.4 cell basis、decomposition frame、measurement basis 的三层措辞

标准 QM 形式常常把叠加写作 $|\psi\rangle = \sum_i c_i |\phi_i\rangle$，其中 $|\phi_i\rangle$ 作为基向量。这一形式在 SAE 框架内需要谨慎处理——$|\phi_i\rangle$ 作为「本征态」常常暗含测量基底，而测量基底涉及 P7 的工作。为避免将后续论文的概念预借到本文，使用以下三层措辞区分基底概念：

第一层 · Cell basis（最原生）：

cell 集合 $C$ 本身提供 $\mathbb{C}^C$ 上的自然 basis $\{|c\rangle : c \in C\}$。ψ 在这组 cell basis 下的展开：

$$\Psi = \sum_{c \in C} z_c |c\rangle$$

cell basis 是 cell-indexed assignment 的 native expression，不暗含任何测量框架。Cell basis 由 cell 集合的 discrete structure 直接给出。

第二层 · Decomposition frame（数学便利）：

数学上选择一组模式展开 $\{|\phi_i\rangle\}$ 来表达 ψ：

$$\Psi = \sum_i c_i |\phi_i\rangle$$

在本文中，$\{|\phi_i\rangle\}$ 被称为 decomposition frame——这是一组数学上的 mode basis，用于把 cell-indexed assignment 分解为不同 mode component。Decomposition frame 是数学 articulation tool，不预设任何物理测量。

Decomposition frame 的具体例子包括：

  • 位置表象 frame：$|\phi_i\rangle = |x_i\rangle$（位置 basis）
  • 动量表象 frame：$|\phi_i\rangle = |p_i\rangle$（动量 basis）
  • 能量本征 frame：$|\phi_i\rangle = |E_i\rangle$（Hamiltonian 本征基底）

decomposition frame 是指称这些 basis 的中性词，不预设它们是测量本征态。

第三层 · Measurement basis（P7 辖区）：

测量基底是 4DD ρ-AND 闭合事件中由测量 apparatus 指定的 closure 通道。Measurement basis 的本体身份由 P7（测量本体身份）处理：apparatus 触发 4DD ρ-AND swap-class event，事件的 closure 通道决定 ψ 在 measurement basis 下的展开如何 collapse 为单一结果。

measurement basis 的本体身份不在本文范围内；本文工作于 decomposition frame 层级。

三层措辞的实际使用：

本文在使用基底相关术语时遵循以下原则：默认使用 cell-indexed assignment $\Psi: C \to \mathbb{C}$ 作为 ψ 的 native object；需要接通标准 QM 形式时，使用 decomposition frame $\{|\phi_i\rangle\}$ 这一中性词；不使用 "measurement basis"、"observable eigenstate"、"projection onto eigenspace" 这些涉及 P6 / P7 工作的术语。这一区分让读者清楚哪些 articulation 在本文范围内，哪些留给后续论文。

本节给 cell-indexed assignment 到 Hilbert space formalism 的前置桥梁，但不完整推出 Hilbert space。

$$\langle \Psi_1, \Psi_2 \rangle = \sum_{c \in C} \overline{\Psi_1(c)} \cdot \Psi_2(c)$$

§7.2 SAE 读法：非分配律作为 ρ-OR 共闭合结构的形式影子

在前闭合 ρ-OR realm 中，$B \lor C$ 不是「$B$ 真或 $C$ 真」的确定回答，而是 $B$ 与 $C$ 在前闭合层级的真实并存。

闭合事件的触发条件（apparatus-level probing 是否必要、是否存在 spontaneous closure mechanism 等）由 P7 处理；本文不在 closure trigger mechanism 上选边。本文需要 anchor 的只是以下两点：

  • 经典分配律左右两边对应不同 closure 状态。左边的 $A \land (B \lor C)$ 在 ρ-OR realm 内保持 $B$、$C$ 多容并存的完整结构；右边的 $(A \land B) \lor (A \land C)$ 隐含 $B$、$C$ 的真值已分别决定——这要求两个独立的完全闭合事件已经发生。

因此经典分配律的左右两边在 ρ-OR realm 中本体上是不同事件——左边保持并存，右边预设决定性回答。这不是逻辑"错了"，而是经典分配律隐含的"完全闭合 + 确定回答"前提在前闭合 ρ-OR realm 中不成立。这是 P1 §3.4 BvN 非分配律债的本体起源 articulation。

§7.3 子空间 lattice 与非分配律最小反例

具体的代数表达：在复数 Hilbert 空间的子空间 lattice 上，非分配律可以通过一个最小例子展示。

考虑二维复 Hilbert 空间 $\mathbb{C}^2$，设：

  • $A = \text{span}\{e_1\}$（沿 $e_1$ 方向的一维子空间）
  • $B = \text{span}\{e_2\}$（沿 $e_2$ 方向的一维子空间）
  • $C = \text{span}\{e_1 + e_2\}$（沿 $e_1 + e_2$ 方向的一维子空间）

其中 $\{e_1, e_2\}$ 是 $\mathbb{C}^2$ 的正交基。

在子空间 lattice 上，$\lor$ 对应 span（线性组合），$\land$ 对应 intersection：

$$B \lor C = \text{span}\{e_2, e_1 + e_2\} = \text{span}\{e_1, e_2\} = \mathbb{C}^2$$

$$A \land (B \lor C) = A \land \mathbb{C}^2 = A = \text{span}\{e_1\}$$

另一边：

$$A \land B = \text{span}\{e_1\} \cap \text{span}\{e_2\} = \{0\}$$

$$A \land C = \text{span}\{e_1\} \cap \text{span}\{e_1 + e_2\} = \{0\}$$

$$(A \land B) \lor (A \land C) = \{0\} \lor \{0\} = \{0\}$$

因此：

$$A \land (B \lor C) = \text{span}\{e_1\} \neq \{0\} = (A \land B) \lor (A \land C)$$

分配律失效。

§7.3 的子空间 lattice 非分配律是数学事实——在任何复 Hilbert 空间的子空间 lattice 上都成立，与 SAE 框架无关。

SAE 框架给这一数学事实的本体读法：

  • 子空间 $A, B, C$ 对应 cell-aggregate 上的 dualization 子模式（不同 dualization 状态的并存配置）
  • $B \lor C$（span）对应「$B$ 与 $C$ 在前闭合层级真实并存」
  • $A \land (B \lor C)$（与并存的 intersection）对应「$A$ 与并存配置的 ρ-OR 共闭合可能性」
  • $A \land B$、$A \land C$（与各分量的 intersection）对应「$A$ 与各分量分别的 ρ-OR 共闭合可能性」
  • $(A \land B) \lor (A \land C)$ 对应「先分别考虑 $A$ 与 $B$、$A$ 与 $C$ 各自的可能性，再 span」

经典逻辑中 $A \land (B \lor C) = (A \land B) \lor (A \land C)$ 因为各步骤都建立在确定真值之上。

§9.2 Fourier dual 在 L₂ 层的本体身份

  • 位置 basis $\{|x\rangle\}$（1DD 标识 layer）与动量 basis $\{|p\rangle\}$（2DD 加性 layer）是同一 cell 上 1DD + 2DD 双层结构的两种 representation
  • 两个 representation 之间的关联通过 L₁→L₂ dualization 实现——dualization 关系自然给出 Fourier-type 复指数关联结构
  • 复指数形式 $e^{ipx/\hbar}$ 中的 $i$ 是 L₁→L₂ algebra remainder（同 §3.2 中复振幅的 $i$）；$ipx/\hbar$ 中的 $\hbar$ 涉及 DD breakthrough 代价（具体身份由 P3 处理）
