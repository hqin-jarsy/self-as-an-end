§2. Definition of the Two-Dimensional Remainder Measure

2.1. Coordinates

For primes p in dyadic block I_j = [2^j, 2^{j+1}), define two coordinates:

Position coordinate u: u_p = rank_{I_j}(p)/m_j ∈ [0,1], where rank is p's ordinal in the block's prime sequence, m_j is total prime count.

Branch gap coordinate g: g(p-1) = ρ_add(p-1) - ρ_mult(p-1), where ρ_add(p-1) = ρ(p-2)+1 (additive path cost), ρ_mult(p-1) = min_{a·b=p-1}[ρ(a)+ρ(b)+2] (optimal multiplicative cost).

2.2. Three Matrices

Discretize u into n_u bins (default 20), g truncated to [-5, +10] (16 bins):

Note: All observed gaps in the tested range fall within this interval; any out-of-range gap is merged into the endpoint tail bin, ensuring every p ∈ I_j is counted in exactly one (u,g)-cell. The zero-mode identity in §2.3 is therefore exact.

Count matrix: N_j(u,g) = #{p ∈ I_j : u_p ∈ bin(u), g(p-1) = g}

Signed mass matrix: M_j(u,g) = Σ_{p ∈ bin(u,g)} h(p)/p

Mean mass matrix: x̄_j(u,g) = M_j(u,g)/N_j(u,g)

with M = N ⊙ x̄ (Hadamard product).

2.3. Zero-Mode Identity

B_j = Σ_{u,g} M_j(u,g). Defining gap marginal signed mass Γ_j(g) = Σ_u M_j(u,g), we have B_j = Σ_g Γ_j(g). This is an identity (coordinate expansion), not a reduction.

Define near-boundary/tie-side and far-multiplicative-side flux: Γ⁺_j = Σ_{g≤1} Γ_j(g) (near-boundary/tie-side aggregate; dominant positive flux from g = 0, 1; small far-additive negative contributions are included but negligible), Γ⁻_j = Σ_{g≥2} Γ_j(g) (far-multiplicative side, persistently negative). Then B_j = Γ⁺_j + Γ⁻_j.

§7. Surrogate Three-Layer Separation

7.1. Design

Null 1: Permute h(p)/p within each gap class. Preserves Γ(g) marginals, breaks within-class u-landscape.

Null 2: Randomize gap labels within each u-bin. Preserves position landscape, breaks gap structure.

Null 3: Replace h(p)/p with i.i.d. Gaussian of same variance. Baseline.

7.3. Three-Layer Interpretation

Null 1's Γ⁺/Γ⁻ is identical to REAL (exact to last digit, all trials, all j). Reason: within-class permutation does not change class sums. Therefore Γ⁺ + Γ⁻ = B_j is an identity — "β-charge pinning" restates 59.1 in gap coordinates, it does not reduce it.

Branch-Orthogonality shows fluctuation does not come from branch-position coupling. The genuine difficulty lies in the mean mass x̄(u,g)'s two-dimensional structure and Mode 2's zero-mode.

§2. 二维余项测度的定义

2.1. 坐标系

对dyadic block I_j = [2^j, 2^{j+1})内的素数p，定义两个坐标：

位置坐标u： u_p = rank_{I_j}(p) / m_j ∈ [0,1]，其中rank是p在block内素数序列中的编号，m_j是block内素数总数。

Branch gap坐标g： g(p-1) = ρ_add(p-1) - ρ_mult(p-1)，其中ρ_add(p-1) = ρ(p-2) + 1（加法路径成本），ρ_mult(p-1) = min_{a·b=p-1}[ρ(a)+ρ(b)+2]（乘法路径最优成本）。g < 0：加法更优。g > 0：乘法更优。g = 0：tie。

2.2. 三个矩阵

将u离散化为n_u个bin（默认n_u = 20），g截断到[G_min, G_max] = [-5, +10]（16个bin），定义：

注： 实验中所有观测到的gap均落入该区间；若出现区间外gap，则合并入端点tail bin，保证每个p ∈ I_j被且仅被计入一个(u,g)-cell。因此§2.3的零模恒等式为精确恒等式。

Count matrix： N_j(u,g) = #{p ∈ I_j : u_p ∈ bin(u), g(p-1) = g}

Signed mass matrix： M_j(u,g) = Σ_{p ∈ bin(u,g)} h(p)/p

Mean mass matrix： x̄_j(u,g) = M_j(u,g) / N_j(u,g)

其中M = N ⊙ x̄（Hadamard积）。

2.3. 零模恒等式

B_j = Σ_{u,g} M_j(u,g)

定义gap marginal signed mass：

Γ_j(g) = Σ_u M_j(u,g)

则B_j = Σ_g Γ_j(g)。这是恒等式（坐标展开），不是reduction。

定义near-boundary/tie侧和far-multiplicative侧通量：

Γ⁺_j = Σ_{g≤1} Γ_j(g)（near-boundary/tie侧aggregate，主正通量来自g=0和g=1；少量far-additive负贡献并入但量级很小）

Γ⁻_j = Σ_{g≥2} Γ_j(g)（far-multiplicative侧aggregate，恒负）

则B_j = Γ⁺_j + Γ⁻_j。

§6. SVD零模分解

6.1. 零模恒等式

设M_j的SVD为M_j = Σ_r s_r a_r(u) b_r(g)^T。则：

B_j = Σ_{u,g} M_j(u,g) = Σ_r s_r · (Σ_u a_r(u)) · (Σ_g b_r(g))

定义每个mode的零模贡献B^(r) = s_r · (Σ_u a_r) · (Σ_g b_r)。

§7. Surrogate三层分离

7.1. 实验设计

三个Null假设：

Null 1： 在每个gap class内permute h(p)/p值。保持β(g)边际，破坏u-landscape内部结构。

Null 2： 在每个u-bin内随机化gap标签。保持位置景观，破坏gap结构。

Null 3： 完全随机高斯替换h(p)/p（same σ）。Baseline。

7.2. 结果

7.3. 三层解读

Null 3 → Null 2（s₁/s₂从~1.3到~5-8）： Gap分布非均匀性的贡献。不同gap class有不同素数比例（83%/17%），这本身产生moderate rank-one。

Null 2 → Null 1（s₁/s₂从~5-8到~30-258）： β(g) profile的贡献。每个gap class有不同的mean h/p——近边界正，深乘法负——这是rank-one的主要来源。

Null 1 → REAL（s₁/s₂从~30-258下降到19-53）： IC递推在marginal rank-one之上添加了结构性rank-2+修正。真实IC数据比纯边际更rich——有gap与position的joint dependence。但这些修正的零模小（§6）。

7.4. 关键发现

Null 1的s₁/s₂高于REAL。 IC递推不是"制造rank-one"，而是在marginal rank-one baseline上引入结构性高阶修正。幸运的是这些修正的零模贡献受控。

Null 1的Γ⁺/Γ⁻与REAL完全一致（精确到最后一位，所有trial，所有j）。原因：gap class内permute不改变class sum。因此Γ⁺ + Γ⁻ = B_j是恒等式——"β-charge pinning"是59.1的gap坐标restating，不是reduction。

9.2. 分裂比例

dB_near/dB_total = 83.3%（近边界素数计数比例）

dB_far/dB_total = 16.6%（远乘法素数计数比例）

§10. 与SAE公理的candidate对应

10.1. 定位

本文是ZFCρ系列中首次系统地将SAE核心概念映射到具体可计算数学结构的工作。以下对应作为candidate提出，不作为公理证明。

10.2. Via Negativa / 非

IC的min操作生成branch-gap坐标g = ρ_add - ρ_mult。这是min操作自由边界的normal coordinate。§4的Γ̃(g) profile是这个自由边界的渐近稳定统计轮廓。

10.3. Via Rho / 余

余项h(p)/p不是误差，而是signed mass。Paper 64的核心对象不是B_j，而是二维余项测度M_j(u,g)。这就是"余项作为测度"的具体数学实现。

10.4. 非与余的操作层配对

Paper 00（余之道）的核心命题："公理层上非独一，操作层上非与余成对。"

在IC递推中：公理层只有min操作（非）；运行后自然产生g坐标（free boundary）和h(p)/p值（remainder mass）——两个因子。

Branch-Orthogonality（count matrix rank-one，§3）说明在population层面，branch type与block位置几乎完全独立——两条Via在population层分离。但mean mass x̄(u,g)的弱rank-one（s₁/s₂ ≈ 3-6，§5）说明在individual remainder amplitude层面，gap与position有genuine coupling。

因此Paper 00的"操作层配对"在count层面得到strong support，在mean层面显示出更复杂的耦合结构。

10.5. λ calibration与gauge

λ = 3.857的选择使全局Σh(p)/p = 0——这是Paper 00所说"非给余以存在条件"的gauge-fixing操作。59.1是这个gauge fixing在finite block上的residual精度问题。

§12. 结论

Paper 64定义了二维余项测度M_j(u,g)并通过SVD分解和Surrogate tests解剖了B_j零模的内部组成。

经过Surrogate淬炼后的五个发现：

(i) Branch-Orthogonality（§3）： Count matrix s₁/s₂从55增长到321。Branch type与block位置的mutual information从4.4e-4降到7.2e-6。这是Paper 63 §3.5经验命题的definitive定量确认。

(ii) 自由边界轮廓（§4）： 归一化Γ̃(g)跨j收敛。相邻j的L1距离从0.023单调降至0.006。Sign flip稳定在g ≈ 1.5。min操作自由边界有渐近稳定的统计结构。

(iii) Count vs Mean（§5）： Signed mass的rank-one = count的极端rank-one（321）× mean的弱rank-one（3-6）。IC递推的genuine二维结构在mean mass里。

(iv) 零模分解（§6）： Mode 1主导B_j，mode 2贡献√(jD)的20-40%，mode 3+可忽略。B_j不是纯rank-1问题。

(v) λ = gauge shift（§9）： dB/dt = -0.6931 ≈ -ln 2，分裂按素数计数比例。λ是uniform calibration。

本文不证明或化简59.1。 B_j = Γ⁺ + Γ⁻是坐标展开不是reduction——Γ⁺/Γ⁻ balance就是B_j smallness的gap坐标restating。

本文提供的是： B_j零模的natural坐标系。在这个坐标系中，B_j来自tie/near-boundary侧（g ≤ 1，主正通量来自g=0和g=1）和深乘法侧（g ≥ 2，负回流）之间的精细对消。λ calibration保证全局对消；finite block fluctuation是59.1。Branch-Orthogonality说明fluctuation不来自branch-position coupling。IC递推的genuine难点在mean mass x̄(u,g)的二维结构和mode 2零模。
