§1 第三个A类公式：m_p·α/m_e

$$\frac{m_p \cdot \alpha}{m_e} = \frac{67}{5} - \frac{1}{1053\left(1+\frac{4}{9}\alpha\right)}$$

§2.2 壳层容量的DD因子分解

Subshell容量（给定n和l）：

$$\text{cap}(n,l) = 2(2l+1) = n_{dual} \times (2l+1)$$

Shell容量（给定n）：

$$\text{cap}(n) = 2\sum_{l=0}^{n-1}(2l+1) = 2n^2$$

DD读法：2来自n_dual（A1公理，Pauli排斥 = L/R不相容）。n不是直接的乘法因子——它是"允许的l值的个数"（l = 0到n−1共n个）。(2l+1)来自SO(3)的角向简并度（3DD乘法结构，因d=4→3维空间→角动量量子化）。

  • 3DD（2l+1）：每种空间折叠的角向简并度

§3 Madelung规则的DD推导

§3.1 n是加法，l是乘法

| 量子数 | DD操作 | DD层 | 含义 |

|--------|--------|------|------|

| n | 加法 | 2DD | 壳层叠加 |

| l | 乘法 | 3DD | 空间折叠 |

| n+l | 总和 | 2DD-3DD边界 | 总拓扑成本 |

n是加法的证据：DD壳层容量s_k = 2k是等差数列。l是乘法的证据：l描述球面驻波模式的折叠深度。

§3.2 Madelung规则 = DD拓扑成本最小化

$$n + l = \text{2DD加法成本} + \text{3DD乘法成本} = \text{总拓扑成本}$$

电子优先填充总拓扑成本最低的轨道。n+l相同时n小的先填——因为1DD硬件（壳层）比3DD软件（空间折叠）更基础。

Madelung规则至今缺少一个被普遍接受的、紧凑而闭式的第一性原理推导。SAE的解释不需要经验输入。

§3.3 2l+1 = DD基本数序列

| l | 轨道 | 2l+1 | DD身份 |

|---|------|------|--------|

| 0 | s | 1 | trivial |

| 1 | p | 3 | n_axes |

| 2 | d | 5 | n_doublets |

| 3 | f | 7 | n_shells + 1 |

来自SO(3)的表示维度，而SO(3)来自d=4（A0公理）。

§3.4 3→4相变 = Aufbau"乱序"

n ≤ 3时，2DD加法成本主导，填充按n排序。n = 3→4时，3DD乘法成本（l=2, d轨道）开始和2DD加法成本（n=4）竞争。4s(n+l=4)先于3d(n+l=5)填充。

这精确对应SAE的相变窗口Ω ∈ [2.75, 4.01]和ZFCρ Paper 57的z/√j peak在Ω ≈ 3.14。

n+l = 4的成本带开始于Z = 13 = n_EW。DD的EW结构数就是相变的起始原子序数。

§3.5 SAE的Aufbau：零未解释例外

经典化学说"先填完一层再去下一层"→ 遇到4s < 3d叫"异常"→ 需要补丁。SAE说n不是唯一排序量，n+l才是。不存在"先填完一层"的优先权被打破——因为层不是组织原则，总拓扑成本才是。

经典化学：一维排序（n）→ 例外无数。

SAE：二维排序（n, l）按总DD成本n+l + 共振修正 → 零未解释例外。

需要说明的是："零未解释例外"不等于"零例外"。Cr(24)、Cu(29)、Pd(46)等仍然偏离naïve的n+l排序——但在SAE框架中，这些偏离全部被归因于DD结构的局部拓扑共振（d⁵ = n_doublets半满，d¹⁰ = n_dual饱和等），不再是无序的经验波动。对于个别深度异常（如Pd的4d¹⁰5s⁰），其本质是高维核电荷的局域应力在相变窗口边缘引发的DD成本权重干涉。SAE声明的是：一切排布异常在本体论上都归因于底层DD结构的局部拓扑共振——不是无规律，而是更精细的规律。

§3.6 第三行为什么只有8个元素

n=3壳层理论容纳18个电子。但3d(n+l=5)被相变窗口切到第四行。第三行只剩3s(n+l=3)和3p(n+l=4)：2+6 = 8 = n_dual × d。

行长度{2, 8, 8, 18, 18, 32, 32}中每个值出现两次——因为相变把每个新的l值"延迟一行"引入。l=2不在n=3行出现（被推迟到n=4行），l=3不在n=4行出现（被推迟到n=6行）。每次推迟 = 行长度重复一次。

行长度每个出现两次也对应L/R镜像对称（A1公理）。

§9 硬预测：稳定岛不存在

§9.1 标准核物理的预测

标准核物理基于壳层模型外推magic numbers，预测Z≈114-126附近存在"island of stability"——超重核可能有增强的稳定性（半衰期从微秒跳到秒甚至年级别）。

§9.2 SAE的反预测

SAE预测稳定岛不存在。

Magic numbers不是壳层势函数的输出——是DD泄漏通道的饱和序列。差值{6, 12, 8, 22, 32, 44}每一个都是DD硬件的具体层级。DD硬件带宽在Z=82处耗尽对外通道，在Z=108处耗尽全部通道。没有"下一个质子magic number"——因为带宽已经用完。

可证伪的硬预测：

Z > 108不存在任何自持稳定的同位素（半衰期 > 1年）。如果实验发现Z > 108的任何同位素半衰期超过1年，SAE的泄漏通道饱和理论被证伪。

当前实验数据完全符合SAE预测：Fl(Z=114)最稳定同位素半衰期~2.7秒，Og(Z=118)最稳定同位素半衰期~0.7毫秒。

§9.3 Magic numbers的本体论地位

标准模型说"我能拟合magic numbers"。SAE说"我知道magic numbers是什么"。

拟合没有本体论地位——可以无限外推（但不知道对不对）。理解有本体论地位——有硬边界（带宽耗尽点），因此有硬预测。

后验给出数据（magic numbers的数值）。先验给出身份（DD结构的哪一层）。两者合起来给出硬边界和硬预测。单靠后验只能外推（软预测），单靠先验只有结构没有数值。

§10 方法论

§10.0 命题分级

本文的命题按证据强度分为三档：

| 档次 | 内容 | 证据类型 |

|------|------|---------|

| A类：cross-observable structured conjecture | §1 m_p·α/m_e = 67/5 − 1/(1053·(1+4α/9))，1.3 ppb | 高精度数值命中 + DD数乘法闭合 + dressing统一规则 |

| B类：structural interpretation | §2-§4 周期表/Madelung/共振 | DD结构对已知化学规律的统一解释（n+l = 拓扑成本，2l+1 = DD序列，例外 = 共振） |

| C类：hard prediction program | §5-§9 稳定性边界/magic numbers/稳定岛 | DD泄漏通道饱和理论 → 可证伪硬预测（Z>108无自持稳定核） |

A类最硬（ppb级数值），B类是结构解释（零未解释例外），C类是最勇敢也最可证伪的前向押注。三档的证据强度依次递减，但科学价值和可证伪性依次递增。

§10.1 QM和SAE的分工

| | QM解释了 | SAE解释了 |

|---|---|---|

| 氢原子 | 能级E_n（精确解） | 为什么α≈1/137（f=g方程） |

| 多电子原子 | 能量趋势（Hartree-Fock近似） | 为什么d=4（四句否定） |

| 周期表排序 | 未解释（Madelung规则无推导） | n+l = DD拓扑成本 |

| Madelung例外 | 未解释（交换能是唯象） | n_doublets/n_shells+1共振 |

| 稳定性上界 | 数值计算（多体核力） | DD泄漏通道饱和（结构论证） |

SAE不替代QM——在QM给出精确解的地方（氢原子），SAE接受QM的计算。SAE在QM断裂的精确位置补上先验：Madelung规则、例外、稳定性上界。

§10.2 DD壳层 vs 原子壳层

DD壳层（A3公理）：s_k = 2k，{2, 4, 6, 8, 10, 12}，总和42。原子壳层：2n²。DD给2k（线性），原子给2k²（二次方）。差一个因子k = 角动量简并度。DD壳层在1DD共存空间中没有3维容纳，只有线性排列。三维空间是后来才出现的（A2公理）。
